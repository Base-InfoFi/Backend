# Flock.io 프로젝트 컨텍스트

## 개요

FLock.io는 AI 훈련과 가치 정렬을 탈중앙화하는 플랫폼입니다. AI 목표가 대중의 윤리와 사회적 목표와 일치하도록 보장하고, 의사결정을 커뮤니티에 맡기며, 유용성을 최우선으로 합니다.

FLock은 생태계 참여를 방해하는 장벽을 제거합니다. 개발자들이 모델, 데이터, 또는 컴퓨팅을 모듈 방식으로 제공할 수 있게 하여, 커뮤니티에 의해, 커뮤니티를 위해, 커뮤니티의 관리 하에 만들어진 다양한 목적에 맞는 모델들을 생성합니다.

인센티브 기반 플랫폼으로 AI 에이전트 훈련, 파인튜닝, 추론을 민주화합니다. 사용자 데이터 수집을 중단하고, 보상의 공정한 분배와 광범위한 거버넌스를 가능하게 합니다.

블록체인 기술과 AI를 활용하여 FLock은 대규모 데이터셋을 다루기 위한 강력한 환경을 제공합니다.

## 중앙화 문제

중앙화된 AI 개발의 문제점:
- 세계 최대 기업들이 자신의 목표에 따라 AI 개발 궤도를 좌우
- 기업의 편향과 가치가 전 세계적으로 증폭
- 모델 접근 권한 결정권이 중앙화됨
- 가치 정렬이 모델 성능을 저하시킴
- 낮은 공공 참여, 컴퓨팅 파워 접근 제한, 데이터 편향과 부정확성

## 시스템 설계

FLock의 시스템 로직은 세 가지 주요 구성요소로 구성됩니다:

### 1. AI Arena
블록체인 기반 탈중앙화 훈련 플랫폼. 태스크 생성 시 모델이 먼저 훈련되고 검증됩니다.

- 훈련 노드들이 모델을 훈련
- 검증자들이 제출된 모델을 평가하고 점수를 제안
- 검증자들이 점수에 대한 합의에 도달하여 제출된 모델의 순위 결정
- 합의 모델은 FL 클라이언트에 할당될 수 있음

### 2. FL Alliance
참가자들의 로컬 데이터를 사용하여 모델을 추가로 파인튜닝합니다.

- 연합 학습(Federated Learning) 접근 방식 활용
- 수천 명의 참가자가 협력하여 글로벌 모델 훈련
- 데이터 주권 보존: 훈련 과정의 어느 단계에서도 로컬 데이터가 전송되지 않음
- 참가자들이 고유한 로컬 데이터로 훈련된 모델의 가중치를 업로드
- 가중치를 집계하여 최적의 글로벌 모델 구축

### 3. AI Marketplace
훈련되고 파인튜닝된 모델을 호스팅하고 배포하는 플랫폼입니다.

- AI Arena 합의 모델 또는 FL 글로벌 모델을 배포 및 호스팅
- 다양한 애플리케이션에 인터페이스 제공
- 블록체인 네트워크의 가상 머신(VM)과 통합
- 복잡한 ML 모델의 원활한 실행 및 추론 제공
- 실시간, 확장 가능하고 안전한 솔루션 제공

## 블록체인 레이어

### 인센티브화
- 블록체인 레이어가 보상 분배에서 핵심 역할
- 참가자들이 스테이킹을 안전하게 잠글 수 있도록 보장
- 신뢰와 투명성 환경 조성
- 기여도에 따라 보상 할당
- 스마트 컨트랙트를 통한 자동화된 보상 프로세스

### 보안
- 지분 증명(Proof-of-Stake) 메커니즘 채택
- 비잔틴 장애 허용 강화
- 부정 행위나 합의 실패 시 스테이킹된 토큰 손실 위험
- 경제적 억제를 통한 정직한 행동 촉진

### 공격 완화
- **Sybil 공격**: 스테이킹 자산으로 다수의 가짜 신원 생성 어려움, 블라인드 검증 메커니즘, 상위 k1 훈련 노드와 k2 검증자만 보상
- **DoS 공격**: 속도 제한 구현
- **무임승차 공격**: 성능이 낮은 참가자는 보상 없음, AI Arena 합의로 정직한 참가자 보상
- **Lookup 공격**: Dataset A와 B 두 개의 검증 세트 사용
- **FL 모델 중독 공격**: 연합 학습 과정에서 편향되거나 손상된 데이터 사용 방지

## AI 레이어

### AI Arena
- 기존 머신러닝(ML) 모델 훈련 패러다임 지원
- 사용자 기기에서 직접 모델 최적화
- 공개 또는 로컬 데이터 사용
- 일반화 능력과 최종 훈련 모델의 성능 최대화
- 커뮤니티 멤버들이 다양한 공개 또는 로컬 데이터 기여 장려
- 블록체인을 통해 기여자 지속적 참여 및 보상 보장

### FL Alliance
- 연합 학습(FL) 접근 방식 활용
- 수천 명의 참가자가 협력하여 글로벌 모델 훈련
- 데이터 주권 보존
- 모델 집계 컴포넌트로 고유한 로컬 데이터로 훈련된 모델의 가중치 업로드
- 가중치 집계로 최적의 글로벌 모델 구축
- 훈련 태스크 자동화 및 배포 오케스트레이션 컴포넌트 통합

### AI Marketplace
- 훈련되고 파인튜닝된 모델 호스팅
- 블록체인 네트워크의 가상 머신(VM) 내에서 ML 모델 배포 환경
- 복잡한 ML 모델의 원활한 실행 및 추론
- 실시간, 확장 가능하고 안전한 솔루션

## 핵심 가치

- 탈중앙화: AI 개발을 다양한 글로벌 커뮤니티의 관리 하에 둠
- 공정성: 보상의 공정한 분배
- 투명성: 블록체인 기반의 투명한 보상 시스템
- 데이터 주권: 로컬 데이터가 전송되지 않음
- 커뮤니티 거버넌스: 의사결정이 커뮤니티에 의해 이루어짐
